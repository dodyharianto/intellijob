IntelliJob is an AI-powered assistant to help users to find relevant jobs and for self-development purposes. This project mainly uses LangGraph to create an agent with multiple tools that can perform tasks based on user documents (resume and portfolio).

# Features
IntelliJob's initial prototype consists of the following features:
- Document Analyzer (RAG)
- Job Search
- Event Search
